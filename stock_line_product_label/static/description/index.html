Print a product barcode label directly from picking line

About

The problem and solution

The Problem:

To print a barcode label for specific product, one have to go search for the product, select the product of interest; go to the product page; click on print label, type in the done product quantities... Its' obviously not scalable.

The Solution:

Simply click on the barcode icon beside the line product of interest and you get to print the product label with the number of labels automatically default to the done quantities (editable to any quantity of interest)

Features

What this module does

Directly print stock picking line product barcode label

Automatically set label quantity to done quantity of the selected line
