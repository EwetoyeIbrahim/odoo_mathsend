About

The problem and solution

The Problem:

Odoo ships with country and states data for some countries not including Nigeria.

The Solution:

This module adds all Nigerian States and Cities (Local Government Areas) to odoo database and enforces city when Nigeria is selected during contact creation.

Features

What this module does

When Nigeria is selected, the users get to pick one of valid Nigerian States

When a state is selected, the user get to pick from the Local Govenment Areas of the selected state
